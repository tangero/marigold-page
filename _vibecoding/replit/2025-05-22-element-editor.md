Nový Element Editor umožňuje přímé úpravy aplikací bez znalosti kódu, podobně jako v design nástrojích.

## ✨ Co je nového:

### 📝 Přímé editování textu
Jednoduše klikněte na jakýkoli textový element a začněte psát. Funguje to stejně jako v design nástrojích — žádné kódování není potřeba.

### 🎨 Vylepšené editování barev
- Zadejte název barvy, hex kód nebo použijte kapátko
- Okamžité náhled změn
- Intuitivní ovládání s menším počtem kliknutí

### 🛠️ Nové style kontroly
- **Velikost písma** - snadné úpravy
- **Tloušťka písma** - modifikace font weight
- **Zarovnání textu** - ovládání alignmentu
- **Barva pozadí** - editace background color
- **Margin** - přizpůsobení okrajů
- **Padding** - úprava vnitřních okrajů

## 🚀 Jak Element Editor funguje

1. **Aktivujte Element Editor** přímo z chat rozhraní
2. **Klikněte na jakýkoli element** v náhledu aplikace
3. **Přímo editujte text** pokud je to string ve zdrojovém kódu
4. **Upravte vlastnosti** jako padding, barva textu, background pomocí intuitivních ovládacích prvků
5. **Okamžitý náhled** změn živě v aplikaci
6. **Uložte a publikujte** okamžitě

### Úspora kreditů
Pokud jsou úpravy přímé, vaše změny se aktualizují přímo ve zdrojovém kódu bez spotřeby kreditů. Pro větší změny Element Editor pošle metadata do Agent.

## 🎯 Výhody

- **Intuitivnější** - systém funguje jako běžné aplikace
- **Rychlejší editace** - změny za sekundy místo minut
- **Real-time aktualizace** - okamžitý náhled při psaní
- **Plynulejší zážitek** - méně přerušení v práci

Element Editor je další krok k tomu, aby tvorba aplikací byla tak přirozená jako používání oblíbeného design nástroje.
