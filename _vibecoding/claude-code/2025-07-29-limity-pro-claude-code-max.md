title: Anthropic omezí Claude Code Max novými týdenními limity

Společnost Anthropic, vývojář konverzačního asistenta Claude, zavede od 28. srpna 2025 nové týdenní limity používání pro předplatitele. Opatření má podle firmy zajistit spravedlivější přístup ke kapacitě systému a řešit problémy s porušováním pravidel používání služby.

Nové omezení se dotkne méně než pěti procent současných uživatelů na základě aktuálních vzorců používání. Anthropic jako hlavní důvod uvádí nebývalý růst nástroje Claude Code, který je součástí předplatitelského balíčku, a identifikované případy porušování pravidel včetně sdílení účtů a prodeje přístupu třetím stranám.

## Struktura nových omezení

Anthropic zachová stávající pětihodinové limity pro jednotlivé relace a přidá dvě nové úrovně týdenních omezení. Každý týdenní limit se obnovuje každých sedm dní nezávisle na pětihodinových cyklech.

Uživatelé tarifu Max 5x mohou podle Anthropic očekávat 140 až 280 hodin používání modelu Claude Sonnet 4 týdně. Pro výkonnější model Claude Opus 4 budou mít k dispozici 15 až 35 hodin týdně. Konkrétní hodnoty se liší podle individuálních vzorců používání.

Těžcí uživatelé modelu Opus pracující s rozsáhlými kódovými bázemi nebo provozující více instancí Claude Code současně dosáhnou svých limitů dříve než běžní uživatelé. Firma upozorňuje, že může hodnoty limitů upravovat podle toho, jak vývojáři Claude Code využívají.

## Důvody zavedení omezení

Claude Code, který umožňuje vývojářům delegovat úkoly programování přímo z příkazové řádky, zaznamenal podle Anthropic nebývalý růst. Současně firma identifikovala problematické vzorce používání, které ovlivňují kapacitu systému pro všechny uživatele.

Mezi problematické praktiky patří pokročilé způsoby využití, kdy uživatelé nechávají Claude běžet nepřetržitě 24 hodin denně na pozadí. Anthropic také řeší případy sdílení účtů mezi více uživateli a komerční prodej přístupu k Claude třetím stranám, což porušuje podmínky služby.

## Dopad na uživatele

Většina uživatelů podle Anthropic nepozná žádný rozdíl, protože týdenní limity jsou navrženy tak, aby podporovaly typické denní používání napříč projekty. Omezení se primárně dotkne uživatelů s mimořádně vysokou spotřebou, kteří využívają Claude Code pro rozsáhlé nebo kontinuální úlohy.

Firma připouští, že ve stejném období uživatelé zažívali několik problémů se spolehlivostí a výkonem služby. Anthropic uvádí, že na opravě těchto problémů aktivně pracuje a bude je řešit v následujících dnech a týdnech.

Společnost se zavazuje, že v budoucnu podpoří dlouhodobé případy použití prostřednictvím jiných možností, ale do té doby budou týdenní limity pomáhat udržovat spolehlivou službu pro všechny uživatele. Předplatitelé mohou své předplatné kdykoli spravovat nebo zrušit v nastavení účtu.
