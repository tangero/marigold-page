Společnost StackBlitz vypustila [Bolt.new v2](https://bolt.new/blog/bolt-v2), aktualizaci nástroje pro tvorbu webových aplikací pomocí AI. Verze přináší rozšíření o backendové funkce, databáze a možnost nasazení aplikací.

## Backendové funkce a databáze

Druhá verze Boltu přidává podporu serverové části aplikací. Vývojáři mohou vytvářet API pomocí Node.js s frameworky Express nebo Fastify. Systém umožňuje definovat endpoints, middleware a zpracování požadavků přímo v AI prostředí.

Bolt v2 integruje databázové systémy Supabase a Turso. Uživatel může vytvořit databázové schéma, definovat tabulky a napojit je na aplikaci bez opuštění prostředí. AI navrhne strukturu databáze podle zadání a vygeneruje potřebný kód pro připojení.

## Deployment a sdílení

Nová verze přidává přímé napojení na platformu Netlify. Aplikaci lze nasadit jedním kliknutím, přičemž Bolt automaticky nakonfiguruje build proces aEnvironment proměnné. Po nasazení získá aplikace veřejnou URL adresu.

Bolt také vylepšil možnosti sdílení projektů. Uživatelé mohou exportovat kód do GitHub repository nebo sdílet živou verzi aplikace s ostatními.

## Prompt Enhancer a pokročilé funkce

Verze 2 zahrnuje režim Prompt Enhancer, který upravuje vstupní zadání pro lepší výsledky. Systém doplní technické detaily, upřesní požadavky a navrhne vhodnou architekturu aplikace.

Bolt nyní podporuje pokročilejší frontend frameworky včetně React Server Components a Astro. Přidává také nástroje pro testování a debugging přímo v prostředí.

## Technické parametry

Bolt v2 používá model Claude Sonnet 4.5 od Anthropic pro generování kódu. Prostředí běží ve WebContainer runtime od StackBlitz, což umožňuje spouštět Node.js přímo v prohlížeči.

Zdarma lze vytvořit omezený počet projektů měsíčně. Placený tarif Pro za 20 dolarů měsíčně nabízí neomezené projekty a rychlejší generování.

## Srovnání s konkurencí

**v0 od Vercel** se zaměřuje primárně na frontend s React komponentami. Podporuje shadcn/ui a Tailwind CSS, ale backend a databáze musí uživatel řešit samostatně. Integruje pouze deployment na Vercel.

**Lovable** nabízí podobný rozsah funkcí jako Bolt v2 - frontend, backend i databáze. Používá vlastní AI modely a podporuje deployment na více platforem. Cena začíná na 20 dolarech měsíčně.

**Replit Agent** umožňuje tvorbu kompletních aplikací včetně backendu a databází. Přidává collaborativní funkce a hosting přímo na Replit infrastruktuře. Agent je dostupný od 25 dolarů měsíčně.

**Cursor a Windsurf** jsou primárně IDE s AI asistencí, nikoli webové prostředí pro tvorbu aplikací. Nabízejí větší kontrolu nad kódem, ale vyžadují lokální vývojové prostředí. Deployment musí uživatel nastavit sám.

Bolt v2 se pozicí a funkcemi podobá Lovable - oba nástroje pokrývají celý stack webové aplikace včetně nasazení. Rozdíl je ve výběru AI modelů a deployment platforem.​​​​​​​​​​​​​​​​
