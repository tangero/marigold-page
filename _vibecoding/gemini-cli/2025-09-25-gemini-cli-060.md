Google představil novou verzi svého open-source AI agenta pro příkazovou řádku - Gemini CLI 0.6.0. Tento týdenní update přináší několik významných funkcí, které zlepšují produktivitu vývojářů pracujících s AI asistencí přímo v terminálu.

## Klíčové novinky verze 0.6.0

Nejpodstatnější změnou je zavedení parametru `--output-format json`, který umožňuje strukturovaný výstup při automatizovaném používání nástroje. Tento formát zahrnuje nejen odpověď modelu, ale také statistiky a případné chyby, což výrazně usnadňuje zpracování v skriptech a dalších nástrojích.

Významnou novinkou jsou také nové klávesové zkratky pro aktivaci takzvaného YOLO režimu. Kombinace `Shift+Y` nebo `Shift+Tab` nyní automaticky potvrzuje všechny čekající dialogy, což výrazně zrychluje práci při opakovaných úlohách, kde uživatel důvěřuje návrhům AI agenta.

Pro zlepšení uživatelského komfortu byly přidány dvě důležité zkratky známé z textových editorů. `Ctrl+R` umožňuje znovu spustit poslední příkaz, zatímco `Ctrl+Z` a `Ctrl+Shift+Z` poskytují funkcionalitu undo a redo pro provedené operace.

## Technické vylepšení

Verze 0.6.0 obsahuje také řadu technických vylepšení pod pokličkou. Byla optimalizována podpora pro `.geminiignore` soubory v glob nástroji, což umožňuje přesnější kontrolu nad tím, které soubory má AI agent zpracovávat. Vylepšena byla také efektivita vyhledávání souborů pomocí algoritmu BFS (Breadth-First Search).

Významnou změnou je refaktoring sémantických tokenů v uživatelském rozhraní a přidání telemetrie pro sledování využívání různých výstupních formátů. Tyto změny pomáhají vývojářům nástroje lépe porozumět, jak uživatelé Gemini CLI využívají.

## O Gemini CLI

[Gemini CLI](https://github.com/google-gemini/gemini-cli) je open-source projekt pod licencí Apache 2.0, který poskytuje přímý přístup k modelům Gemini 2.5 Pro přímo v terminálu. Nástroj využívá kontextové okno o velikosti jednoho milionu tokenů, což mu umožňuje analyzovat celé kódbázy najednou.

Bezplatná úroveň nabízí 60 požadavků za minutu a 1000 požadavků denně s osobním Google účtem. Pro pokročilejší využití podporuje autentizaci přes Google Cloud Console API klíče nebo Vertex AI.

Nástroj je součástí širšího ekosystému [Google Cloud AI nástrojů](https://cloud.google.com/gemini/docs/codeassist/gemini-cli) a sdílí technologie s Gemini Code Assist, který poskytuje AI asistenci ve VS Code a dalších IDE.

## Dostupnost a instalace

Gemini CLI 0.6.0 je nyní dostupný ke stažení na [GitHub releases stránce projektu](https://github.com/google-gemini/gemini-cli/releases/tag/v0.6.0). Google plánuje vydávat nové preview verze každý týden v úterý ve 23:59 UTC, což umožní rychlejší testování nových funkcí komunitou vývojářů.

Projekt aktivně vítá příspěvky od komunity a má veřejně dostupný [roadmap](https://github.com/google-gemini/gemini-cli) pro verzi 1.0. Vývojáři mohlí hlásit chyby, navrhovat funkce a přispívat kódem přímo prostřednictvím GitHub repozitáře.
