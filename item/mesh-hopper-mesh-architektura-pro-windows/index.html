Mesh hopper – mesh architektura pro Windows

Patrick Zandl · 23. květen 2003

Firma BPO Solutions představila na londýnském WLAN Event svoji mesh architekturu pro Windows platformu. Jmenuje se to Mesh hopper a mělo by to umožnit každému Windows počítači zapojit se a spoluvytvářet mesh síť - podporuje to všechny 802.11 deriváty a vypadá to opravdu moc slibně. Pokud nevíte, co je mesh síť, přečtěte si tento článek na Marigoldovi, je to zajímavá věc umožňující každému vytvářet vlastní síť na bázi P2P a předávat provoz od jednoho počítače k druhému. ČTÚ se všemi licencemi jistě bude mít radost, až to k nám dorazí.

Více podrobností není o Mesh hopperu známo, pár marketingových slintů je na webu služby, ale jak to pořádně funguje nevím - Jullian reportoval z Londýna, že na stánku chytil jen hostesky a dostal jen papíry s obecným povídáním a tak to zkusí ještě dnes, snad tam chytí technika...

Na mesh sítě se těším, zatím všechna řešení byla docela těžkopádná, nebo obtížně dostupná a drahá. Přitom kouzlo mesh sítě je v jednoduchosti a dostupnosti i pro BFU - jen tak lze dosáhnout masovosti, díky níž bude mesh síť reálnou mobilní sítí!
