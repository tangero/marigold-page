## Souhrn

Microsoft vydal volitelnou aktualizaci KB5067036 pro Windows 11 verze 24H2 a 25H2, která přináší dvě významné novinky: bezpečnostní funkci Administrator Protection a přepracované Start Menu. Aktualizace je součástí měsíčního testovacího cyklu a neobsahuje bezpečnostní záplaty.

## Klíčové body

- Aktualizace KB5067036 zavádí funkci Administrator Protection, která snižuje riziko spuštění škodlivých příkazů s administrátorskými právy
- Redesignované Start Menu přináší nové kategorie, mřížkové zobrazení, responzivní rozvržení a posuvnou sekci "Vše"
- Aktualizace opravuje chyby v Media Creation Tool a problémy s HTTP/2 připojeními na localhost
- Jedná se o volitelnou aktualizaci, která aktualizuje Windows 11 24H2 na build 26100.5074 a verzi 25H2 na build 26100.7019
- Uživatelé mohou aktualizaci nainstalovat ručně přes Windows Update nebo stáhnout z Microsoft Update Catalog

## Podrobnosti

Aktualizace KB5067036 je součástí pravidelného měsíčního cyklu testovacích aktualizací, které Microsoft vydává na konci každého měsíce. Na rozdíl od běžných aktualizací Patch Tuesday, které vycházejí druhé úterý v měsíci a obsahují bezpečnostní záplaty, tyto preview aktualizace slouží k testování nových funkcí a oprav, které budou následně zahrnuty do příští pravidelné aktualizace.

Funkce Administrator Protection představuje významné bezpečnostní vylepšení systému Windows 11. Tato funkce má za cíl snížit riziko, že škodlivé programy získají administrátorská práva a provedou nebezpečné operace v systému. Jde o další vrstvu ochrany nad rámec standardního User Account Control (UAC), která má zabránit neoprávněnému spouštění příkazů s vysokými oprávněními.

Přepracované Start Menu přináší výraznější změny v uživatelském rozhraní. Nové kategorie a mřížkové zobrazení mají usnadnit orientaci mezi nainstalovanými aplikacemi. Responzivní rozvržení se přizpůsobuje velikosti okna a posuvná sekce "Vše" umožňuje rychlejší procházení všech dostupných aplikací bez nutnosti přepínání mezi různými pohledy.

Aktualizace také řeší konkrétní technické problémy. Oprava Media Creation Tool, nástroje pro vytváření instalačních médií Windows, řeší problémy, které bránily jeho správnému fungování. Druhá oprava se týká HTTP/2 připojení k webovým službám běžícím na localhost, což je důležité zejména pro vývojáře a uživatele lokálních serverů.

Instalace je dobrovolná a uživatelé si mohou vybrat, zda aktualizaci nainstalují okamžitě, nebo počkají na její začlenění do příští povinné aktualizace. Automatická instalace proběhne pouze u uživatelů, kteří mají aktivní možnost "Získat nejnovější aktualizace, jakmile budou k dispozici".

## Proč je to důležité

Funkce Administrator Protection představuje významný posun v bezpečnostní architektuře Windows. V době rostoucího počtu kybernetických útoků, které zneužívají administrátorská práva k šíření ransomwaru a jiného malwaru, přináší tato funkce dodatečnou ochranu pro miliony uživatelů Windows 11. Jde o reakci Microsoftu na měnící se hrozby v kyberprostoru a snahu o implementaci principu nejmenších oprávnění i pro administrátorské účty.

Redesign Start Menu ukazuje, že Microsoft pokračuje v iterativním vylepšování uživatelského rozhraní Windows 11, které bylo při uvedení kritizováno za odstranění některých funkcí z Windows 10. Postupné přidávání funkcí a vylepšování použitelnosti naznačuje, že společnost reaguje na zpětnou vazbu uživatelů.

Z hlediska širšího kontextu jde o součást strategie Microsoftu postupně zavádět nové funkce prostřednictvím volitelných aktualizací, což umožňuje testování v reálném prostředí před jejich plným nasazením. Tento přístup snižuje riziko problémů s kompatibilitou a stability, které v minulosti provázely některé větší aktualizace Windows.
