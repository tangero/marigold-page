## Souhrn

Microsoft dokončuje testování další velké aktualizace pro Windows 11, která přinese zásadně přepracované Start menu poprvé oznámené začátkem roku. Aktualizace je nyní v Release Preview kanálu programu Windows Insider, což je poslední fáze před oficiálním vydáním pro všechny uživatele, které by mohlo přijít již příští měsíc.

## Klíčové body

- Nové Start menu s přepracovaným rozhraním, které upřednostňuje seznam aplikací před doporučeným obsahem
- Kategorizovaný pohled na aplikace podobný iOS/iPadOS s možností přepnutí na klasické abecední řazení
- Vylepšená integrace s Phone Link pro rychlý přístup k fotografie, notifikacím a zprávám ze smartphonu
- Nová ikona baterie na hlavním panelu s možností zobrazit procenta a žlutým varováním při nízkém stavu
- Vylepšení Průzkumníka souborů a další změny kvality života systému

## Podrobnosti

Nové Start menu představuje největší změnu tohoto klíčového prvku Windows 11 od jeho uvedení. Microsoft přesunul seznam aplikací do hlavní oblasti nabídky, čímž reaguje na kritiku uživatelů, kteří považovali původní design za příliš zaměřený na doporučený obsah. Uživatelé nyní mohou vybrat mezi novým kategorizovaným pohledem, který seskupuje aplikace podobně jako na zařízeních Apple, nebo klasickým abecedním seznamem.

Stejně důležitá je nová úroveň přizpůsobení. Uživatelé mohou zcela vypnout sekci s doporučeným obsahem a místo toho připnout více aplikací, nebo dokonce odstranit obě sekce (doporučené i připnuté) a ponechat pouze seznam aplikací. Tato flexibilita reaguje na různé preference uživatelů, kteří měli s původním designem Start menu problémy.

Integrace Phone Link přímo do Start menu umožňuje rychlý přístup k nedávným fotografiím a notifikacím ze smartphonu, možnost odpovídat na textové zprávy nebo zobrazit obrazovku telefonu bez nutnosti otevírat samostatnou aplikaci. Jde o pokus Microsoftu o hlubší propojení Windows s mobilními zařízeními.

Vylepšení hlavního panelu zahrnují přepracovanou ikonu baterie, která je větší, lépe viditelná a mění barvu na žlutou při nízkém stavu. Uživatelé také získají možnost zobrazit procenta baterie přímo na hlavním panelu, což byla dlouho požadovaná funkce.

## Proč je to důležité

Tato aktualizace ukazuje, že Microsoft aktivně reaguje na zpětnou vazbu uživatelů ohledně Windows 11. Původní Start menu bylo často kritizováno za příliš velký důraz na doporučený obsah a nedostatek přizpůsobení. Nový design se vrací k praktičtějšímu přístupu zaměřenému na rychlý přístup k aplikacím.

Zařazení aktualizace do Release Preview kanálu signalizuje, že Microsoft považuje tyto změny za stabilní a připravené k nasazení. Pokud testování proběhne bez problémů, mohou se uživatelé těšit na vydání během příštího měsíce. Jde o důkaz, že Microsoft pokračuje v iterativním vylepšování Windows 11 místo čekání na velké roční aktualizace.
