## Souhrn

Microsoft dokončuje testování velké aktualizace Windows 11, která přinese kompletně přepracované Start menu, vylepšení hlavního panelu a Průzkumníka souborů. Aktualizace je nyní v Release Preview kanálu Windows Insider programu, což je poslední fáze před oficiálním vydáním pro všechny uživatele, které by mohlo přijít již příští měsíc.

## Klíčové body

- Nové Start menu s přepracovaným rozhraním, které upřednostňuje seznam aplikací před doporučeným obsahem
- Kategorizovaný pohled na aplikace podobný iOS/iPadOS s možností přepnutí na klasické abecední řazení
- Vylepšená ikona baterie na hlavním panelu s větší velikostí, žlutým varováním při nízkém stavu a zobrazením procent
- Integrace Phone Link panelu přímo do Start menu pro rychlý přístup k notifikacím a zprávám ze smartphonu
- Možnost úplného odstranění sekcí Doporučené a Připnuté aplikace pro čistší rozhraní

## Podrobnosti

Nové Start menu představuje zásadní redesign oproti současné verzi. Microsoft přesunul seznam aplikací do hlavní oblasti a nabízí dva režimy zobrazení - nový kategorizovaný pohled, který seskupuje aplikace podobně jako na iOS zařízeních, nebo klasické abecední řazení. Uživatelé získávají větší kontrolu nad vzhledem menu, včetně možnosti vypnout sekci Doporučené a nahradit ji dalšími připnutými aplikacemi, nebo obě sekce zcela odstranit a ponechat pouze seznam aplikací.

Integrace Phone Link představuje zajímavé propojení s mobilními zařízeními. Panel se zobrazuje vedle Start menu a umožňuje prohlížet nedávné fotografie, notifikace a odpovídat na textové zprávy nebo zobrazit obrazovku telefonu bez nutnosti otevírat samostatnou aplikaci.

Hlavní panel dostává vylepšenou ikonu baterie, která je větší a lépe viditelná. Při nízkém stavu nabití se zbarví žlutě a uživatelé mohou zapnout trvalé zobrazení procent baterie přímo na panelu. Průzkumník souborů získává nové funkce, jejichž detaily Microsoft zatím plně neodhalil.

Aktualizace je nyní v Release Preview kanálu, což znamená, že prošla většinou testování a je připravena k nasazení. Pokud se nevyskytnou kritické problémy, běžní uživatelé by měli novinky obdržet během příštího měsíce prostřednictvím standardních Windows Update.

## Proč je to důležité

Tato aktualizace představuje první větší redesign Start menu od vydání Windows 11 v roce 2021. Microsoft reaguje na kritiku uživatelů, kteří považovali původní Start menu za příliš jednoduché a málo přizpůsobitelné ve srovnání s Windows 10. Větší flexibilita a možnost odstranění doporučeného obsahu ukazuje, že Microsoft naslouchá zpětné vazbě.

Integrace Phone Link přímo do systémového rozhraní signalizuje pokračující snahu o propojení Windows s mobilními zařízeními, což je oblast, kde Microsoft historicky zaostával za konkurencí jako Apple s ekosystémem macOS/iOS. Vylepšení hlavního panelu a Průzkumníka souborů pak ukazují, že Microsoft se soustředí na postupné zdokonalování základních prvků systému, což by mělo zlepšit každodenní uživatelský zážitek.
