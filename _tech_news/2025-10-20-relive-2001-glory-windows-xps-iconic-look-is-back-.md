## Souhrn

Vývojář Gorjan Jovanovski vydal bezplatný launcher pro Android, který přináší autentický zážitek z Windows XP na moderní telefony. Aplikace nejen věrně napodobuje vzhled legendárního operačního systému z roku 2001, ale obsahuje i funkční menu Start, klasické aplikace přepsané od základu a podporu pro další retro verze Windows včetně 95, 98, ME a 2000.

## Klíčové body

- Launcher replikuje kompletní prostředí Windows XP včetně zvuků, tapet a funkčního menu Start
- Podporuje přepínání mezi vzhledem Windows 95, 98, ME, 2000 a XP, každý s vlastními uvítacími obrazovkami a zvuky
- Obsahuje přepsané klasické aplikace jako Winamp, Notepad, Paint, Minesweeper nebo Internet Explorer
- Nabízí moderní funkce jako gesta pro ovládání, integraci kalendáře a počasí
- Launcher je dostupný zdarma ke stažení jako APK soubor pro boční načtení

## Podrobnosti

Moderní telefony s Androidem disponují výkonem mnohonásobně převyšujícím stolní počítače z roku 2001, což umožňuje bezproblémové spuštění rozhraní připomínajícího Windows XP. Jovanovski této skutečnosti využil a vytvořil launcher, který jde daleko za pouhé vizuální napodobení.

Aplikace nabízí plně funkční plochu s ikonami, které lze přejmenovávat a měnit, přičemž vývojář zahrnul rozsáhlou sbírku autentických ikon z dané éry. Menu Start funguje jako spouštěč aplikací a launcher podporuje moderní gesta – přejetí dolů zobrazí notifikace, nahoru otevře menu Start a vyhledávání, přejetí doprava spustí oblíbenou aplikaci.

Zvláštní pozornost si zaslouží přepsané retro aplikace. Jovanovski vytvořil od základu funkční verze klasických programů optimalizované pro moderní telefony. Mezi nimi nalezneme Winamp pro přehrávání hudby, Notepad pro psaní poznámek, Paint pro kreslení, Minesweeper a Solitaire pro hraní, Internet Explorer jako webový prohlížeč, nebo dokonce Calculator a Windows Media Player. Launcher dokonce obsahuje legendární 3D Pipes spořič obrazovky.

Aplikace není dostupná v Google Play Store, ale lze ji stáhnout jako APK soubor. Vývojář poskytl i kontrolu přes VirusTotal pro ověření bezpečnosti. Launcher zahrnuje i vlastní systém aktualizací, který kontroluje dostupnost nových verzí na vývojářově serveru.

## Proč je to důležité

Projekt představuje zajímavý příklad toho, jak lze spojit nostalgii s moderními technologiemi. Ukazuje také výkonové možnosti současných mobilních zařízení a kreativitu nezávislých vývojářů. Pro uživatele, kteří vyrůstali s Windows XP, nabízí autentický nostalgický zážitek, zatímco mladší generace může objevit, jak vypadalo počítačové rozhraní před dvěma dekádami. Launcher je zároveň plně funkční a může sloužit jako každodenní rozhraní telefonu, což z něj činí více než jen technologickou kuriozitu.
