## Souhrn

Microsoft oficiálně ukončil standardní podporu Windows 10, ale uživatelům, kteří nemohou nebo nechtějí přejít na Windows 11, nabízí program Extended Security Updates (ESU). Ten zajistí bezpečnostní aktualizace až do 13. října 2026, přičemž registrace je možná zdarma prostřednictvím služby OneDrive nebo Microsoft Rewards.

## Klíčové body

- Windows 10 stále používá 41 % uživatelů PC, přestože Microsoft ukončil standardní podporu
- Program ESU prodlouží bezpečnostní aktualizace do října 2026
- Standardní cena registrace je 30 dolarů, ale existují tři bezplatné možnosti
- Zdarma mohou registrovat Evropané, uživatelé OneDrive Backup nebo ti, kdo vykoupí 1000 bodů Microsoft Rewards
- Nutná je aktualizace na verzi Windows 10 22H2

## Podrobnosti

Microsoft explicitně uvádí, že program ESU není dlouhodobým řešením, ale dočasným mostem k bezpečnému přechodu na novější platformu. Bezpečnostní zranitelnosti Windows představují vážné riziko, proto je program nezbytný pro uživatele, kteří nemohou upgradovat na Windows 11 kvůli hardwarovým omezením nebo jiným důvodům.

Pro registraci do programu musí uživatelé splnit několik podmínek. Především je nutné mít nainstalovanou nejnovější verzi Windows 10 22H2, což lze ověřit v Nastavení pod položkou Windows Update a View Update History. Dále je vyžadováno administrátorské oprávnění k počítači.

Samotná registrace probíhá přes rozhraní Windows Update v Nastavení, kde by se měl zobrazit odkaz s textem o ukončení podpory Windows 10 v říjnu 2025 a možnosti registrace do Extended Security Updates.

Bezplatné možnosti registrace zahrnují použití služby Windows Backup se synchronizací nastavení do OneDrive účtu, nebo vykoupení 1000 bodů z programu Microsoft Rewards. Evropští uživatelé mají registraci zdarma automaticky, pravděpodobně kvůli regulačním požadavkům EU.

## Proč je to důležité

S téměř polovinou uživatelů PC stále na Windows 10 představuje ukončení podpory významné bezpečnostní riziko. Mnoho počítačů nesplňuje hardwarové požadavky Windows 11, zejména požadavek na čip TPM 2.0, což uživatele staví před volbu mezi nákupem nového zařízení nebo provozem nezabezpečeného systému. Program ESU poskytuje praktické řešení pro přechodné období a dává uživatelům další rok na přípravu migrace nebo nákup kompatibilního hardware.
