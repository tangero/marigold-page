NASTAVENÍ PRO CLOUDFLARE PAGES
==============================

Při importu repository v Cloudflare Pages použij tato nastavení:

Framework preset: Jekyll
Build command: jekyll build --config _config_vibecoding.yml && cp vibecoding-index.html _site/index.html
Build output directory: _site
Root directory: (ponech prázdné)

Environment variables:
JEKYLL_ENV: production

Branch deployments:
Production branch: main
Preview branches: Enable all non-production branches

DŮLEŽITÉ:
- Build command obsahuje kopírování vibecoding-index.html jako hlavní index stránky
- Pokud bude problém s build commandem, můžeš ho rozdělit do build skriptu
